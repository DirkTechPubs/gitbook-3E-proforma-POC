# Update Card Purge Type

Do the following to reduce a card's values to zero and updates the Purge Type.

**Note**: If a change was made to the proforma billable amounts and the proforma **Purge** action is selected, the billable amount changes are ignored.

1. Locate the proforma containing funds on the Proforma list.
2. Click the proforma number to access the Proforma Details view.
3. Click the appropriate tab (i.e., Fees, Costs, or Charge)
4. Click the card-level **Action** menu and select **Purge**.
5. Select a purge type and click **Purge**.
6. Click **Save & Close**.
