# Add Adjustments

Do the following to add an adjustment card:

1. Access the Proforma Detail view for a proforma.
2. Click the **Adjustment** tab and click **Add Adjustment**.

**Note**: You can also select Add Adjustment from the Proforma Detail's Action menu.

3. Select an adjustment **Type** and **Method**.
4. Select additional adjustment options as needed. See [Add Adjustment Form and Field Definitions](add-adjustment-form-and-field-definitions.md#add-adjustment-form-and-field-definitions) for further details.
5. Click **Save Adjustments** to apply your settings and close the Add Adjustment form.
