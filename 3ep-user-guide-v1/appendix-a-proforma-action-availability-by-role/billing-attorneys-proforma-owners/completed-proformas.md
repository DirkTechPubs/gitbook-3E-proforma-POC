# Completed proformas

Selecting the **Completed proformas** category displays all proformas for which the user is the owner, and which are successfully submitted.

Proformas with the following statuses are available:

| **Status**    | **Sub-status**                                                                                                                         | **Description**                                                                    |
| ------------- | -------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| **Completed** | -                                                                                                                                      | Proformas in this status have been successfully submitted by the billing attorney. |
| Approved      | Proformas in this sub-status have been successfully submitted by the billing attorney and approved by an approver.                     |                                                                                    |
| Biller review | Proformas in this sub-status have been submitted by the billing attorney but require review and approval of certain edits by a Biller. |                                                                                    |
| Deferred      | Proformas in this sub-status have been successfully deferred.                                                                          |                                                                                    |

&#x20;

The following actions are available depending on proforma status/sub-status:

| **Action**        | **Available with status/sub-status**                                     | **Description**                                                                                                                                                                                             |
| ----------------- | ------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Generate bill** | Urgent/Completed, Completed, Completed/Approved, Completed/Biller review | <p>Select this action when you want to generate an invoice for the proforma.</p><p><strong>Note</strong>: Only the proforma owner with the appropriate rights can generate bills.</p>                       |
| **Reclaim**       | Urgent/Completed. Completed, Completed/Approved, Completed/Biller review | Select this action when you want to return a proforma back to the Needs review category after it was already submitted.                                                                                     |
| **Undefer**       | Completed/Deferred                                                       | <p>Select this action when you want to return a proforma back to the Needs review category after it was deferred.</p><p><strong>Note</strong>: Only the proforma owner can undefer a deferred proforma.</p> |
