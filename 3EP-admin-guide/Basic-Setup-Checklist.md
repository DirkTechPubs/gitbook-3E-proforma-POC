# Basic Setup Checklist

This section provides a basic setup checklist for the settings in 3E that must be set up prior to using 3E Proforma.

Minimum requirements have been noted. Review the full document for information about each step.

1.  [<u>Override/Set System Options</u>](OverrideSet-System-Options.md#overrideset-system-options)

    1.  **Framework** group

        1.  EnableObjectAudit = TRUE

    <!-- -->

    2.  **3E Proforma** group

        1.  Use Billing Workflow = TRUE at Firm level

        2.  ProformaStatus_Closed_Code – set to the firm’s closed proforma status code

        3.  ProformaStatus_Closed_Processed_Code– set to the firm’s closed proforma status code for processed proformas

        4.  ProformaStatus_Defer_Code– set to the firm’s deferred proforma status code

        5.  ProformaStatus_Undefer_Code– set to the firm’s current proforma status code
