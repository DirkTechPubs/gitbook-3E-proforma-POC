### **Global Allow Actions (True, False)**

“Global Allow” means that the actions are used across the application in many locations and there is one option to configure the access for all locations. For example, Print Proforma is available in these locations:

- My Proformas – Bulk Actions Menu – Print Proforma

- My Proformas – Card View – Print button

- My Proformas – Grid View – Actions Menu – Print Proforma

- Proforma Details – Proforma Actions Menu – Print Proforma

For example, the **Global_Allow_PrintProforma** system option controls all four of the print proforma actions above with one system option.

For the “Global Allow” settings, the system default value is “True”. If there is no change, these fields/actions are available to all users.
