### **List Options (Full, Read, Hidden)**

Some fields are editable and have three options

- Full = full access, read/write access

- Read = read only access

- Hidden = item is hidden from the UI

The system default value is “Full” for all options.

Note that you may see a few items with “Full” access option that are not currently editable. These fields will be editable in the future, and we set up the system options to work with the future enhancement. For now, “Full” is the same as “Read” for those fields.
