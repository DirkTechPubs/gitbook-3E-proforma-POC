### **Supporting documentation**

A setup worksheet has been provided in **000035886:** *3E Proforma - System Options Configuration Worksheet* to assist with planning, testing, and documenting your system option setup.

This worksheet contains a full list of all the system options and what they control. Since the default behavior is to enable access, most customers will not need to change many of the options, but it is important to understand which ones you want to use and test their setup.

**Power Move:** Use the filter in the Excel sheet to find specific overrides instead of reading the full list.
