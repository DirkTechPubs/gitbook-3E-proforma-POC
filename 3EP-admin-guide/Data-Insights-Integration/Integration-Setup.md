## Integration Setup

All setup is done in Data Insights. The customer must have the Data Insights 4.10+ version of the Performance Metrics tabular model. The stock DI report is called **3E Proforma Matter Summary**. Detailed Data Insights setup instructions are available in the *Data Insights 4.10 Cloud Release Notes* (KC No. 000005960) and the *3E Data Insights 4.3 On-Premises Installation Guide* (KC No. 000005898).
