# Hybrid Setup

3E Proforma is a cloud module and as such may be run in a hybrid setup with 3E on premises. This hybrid configuration requires that you follow additional setup steps to enable the cloud module(s) to connect to the on premises 3E database. A services engagement is recommended; however, the basic steps are documented in KC article **000005250:** *3E Cloud Modules – Getting Started Guide.*

In addition, it is important to note that 3E Proforma is tightly integrated with 3E and as such many features are dependent upon the 3E and 3E UI software code. If you are currently using an older version of 3E, you may not have access to all the current 3E Proforma features and corrections. Please review KC article **000006964:** *3E Proforma Hybrid: Release Dependencies* for details about 3E and 3E UI version dependencies.
