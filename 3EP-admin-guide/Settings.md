# Settings

The Settings menu <img src="images/media/image43.png" style="width:0.34375in;height:0.36458in" /> contains User Settings, which allow a user to set default behavior for certain areas of the system. These settings are user specific. All users have access to the User settings tab.

If the user also has the **3EProformaAdminRole** assigned to them, they will also have access to the Firm Settings and System Options tabs. Firm settings are default values set at the firm level, user settings will override any firm settings.

Any time a setting is changed, be sure to click the **Save changes** button to save.
