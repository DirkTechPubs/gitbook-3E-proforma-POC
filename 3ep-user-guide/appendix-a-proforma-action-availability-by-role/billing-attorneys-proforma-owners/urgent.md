# Urgent

Selecting the **Urgent** category displays all proformas for which the user is the owner, which are not submitted yet and the due date has passed.

Proformas with the following statuses are available:

| **Status**      | **Sub-status**                                                                                                                                                                                                                                                          | **Description**                                                    |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ |
| **Urgent**      | -                                                                                                                                                                                                                                                                       | Proformas in this status have met or surpassed their urgency date. |
| Rejected        | Proformas in this status have been rejected by the approver and returned to the billing attorney. Their urgency date has been met or surpassed.                                                                                                                         |                                                                    |
| Returned        | Proformas in this status have been returned to the billing attorney. Their urgency date has been met or surpassed.                                                                                                                                                      |                                                                    |
| Biller returned | This status is for proformas that were returned to the billing attorney by a biller after they were already submitted, and the biller approves/makes specified edits and that were opened/edited by the billing attorney. Their urgency date has been met or surpassed. |                                                                    |
| Completed       | Proformas in this status have been already submitted by the billing attorney and are waiting for approver review. Their urgency date has been met or surpassed.                                                                                                         |                                                                    |

The following actions are available depending on proforma status/sub-status:

| **Action**         | **Available with status/sub-status**                                                   | **Description**                                                                                                                                                          |
| ------------------ | -------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Print Proforma** | <p>Urgent/all sub-statuses</p><p> </p>                                                 | Select this action to view a PDF version of the proforma                                                                                                                 |
| **Bill Preview**   | <p>Urgent/all sub-statuses</p><p> </p>                                                 | Select this action to view a PDF version of the bill as it will appear to the client                                                                                     |
| **Close**          | <p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p>                              | <p>Select this action to close the proforma without saving and remove it from the workflow.</p><p><strong>Note</strong>: Close is only available to Proforma Owners.</p> |
| **Submit**         | <p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p><p>Urgent/Biller returned</p> | Select this action to indicate that all the edits for this proforma have been completed and it should be sent to the biller.                                             |
| **Reclaim**        | <p>Urgent/Completed</p><p> </p>                                                        | Select this action to return a proforma back to editing after it was submitted                                                                                           |
| **Forward**        | <p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p><p>Urgent/Biller returned</p> | Select this action to forward a proforma to another user.                                                                                                                |
| **Defer**          | <p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p><p>Urgent/Biller returned</p> | Select this action to indicate that work on this proforma should be deferred to a later time                                                                             |
| **Set priority**   | <p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p><p>Urgent/Biller returned</p> | Select this action to set a proforma as priority. Other users who have the proforma in their list will also see that it is marked as priority.                           |
| **Comments**       | <p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p><p>Urgent/Biller returned</p> | Select this action to view/add proforma-level comments.                                                                                                                  |
