# Filters

Proformas and cards are displayed in lists and grids in 3E Proforma. Filters are available to help reduce the amount of time needed to locate the record(s) you're looking for. Filters are available for:

* Proforma List and Detail View,
* Card lists (i.e., Fees and Costs),
* WIP List
* Archived Proformas

## Accessing filters

Click the Filter icon where available, the filter panel will display.

![](../../.gitbook/assets/image38.png)

## Specifying and Applying filter criteria

Select criteria such as specifying specific client names or matter numbers to include in a list. Additionally, you can filter list based on numerical values. Click **Apply** to save your filter criteria. The applied criteria, displays at the top of the Filter panel.

## Filtering numeric fields

When filtering using numerical values, enter a value and use the available filter operators **Greater than**, **Less than**, or **Equal to**.

## Clearing filters

Do one of the following to clear filters:

* **Clear all criteria** - Click **Clear**, at the bottom of the panel.
* **Remove specific criteria** - Click the **X** on a specific criteria badge displayed at the top of the filter panel.

![](../../.gitbook/assets/image39.png)

## Filter Criteria

The list of proformas can be filtered by the following criteria:

**Note**: Available filter criteria will vary depending on the page or list.

| **Field Name**         | **Description**                                                                                                                                                                                      |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Role**               | Selecting a role in this criterion narrows down the list of proformas to those where the user's role on the proforma is the selected value. Available values are: Approver, Co-Owner, Editor, Owner. |
| **Opened status**      | Selecting an opened status will narrow down the list of proformas to either Unopened or Opened proformas. The number in parenthesis is the number of proformas of that status.                       |
| **Locked by**          | Selecting values in this criterion narrows down the list of proformas locked by selected users.                                                                                                      |
| **Total**              | Selecting Greater than/Less than or Equal to and populating the value in this criterion narrows down the list of proformas by the total amount of the proforma.                                      |
| **Client name**        | Selecting values in this criterion narrows down the list of proformas by selected clients, More than one client can be selected.                                                                     |
| **Client number**      | Selecting values in this criterion narrows down the list of proformas by the selected client number(s). More than one client number can be selected.                                                 |
| **Matter description** | Selecting values in this criterion narrows down the list of proformas by selected matters. More than one matter can be selected.                                                                     |
| **Matter number**      | Selecting values in this criterion narrows down the list of proformas by the selected matter number(s). More than one matter number can be selected.                                                 |
| **Total fees**         | Selecting Greater than/Less than or Equal to and populating value in this criterion narrows down the list of proformas by the total fees of the proforma.                                            |
| **Total Costs**        | Selecting Greater than/Less than or Equal to and populating value in this criterion narrows down the list of proformas by the total costs of the proforma.                                           |
| **Total Charges**      | Selecting Greater than/Less than or Equal to and populating value in this criterion narrows down the list of proformas by the total charges of the proforma.                                         |
| **Bill group**         | Selecting values in this criterion narrows down the list of proformas by selected bill groups. More than one bill group can be selected                                                              |
| **Matter currency**    | Selecting values in this criterion narrows down the list of proformas by selected currency. Only one currency can be selected.                                                                       |
| **Available funds**    | Selecting Greater than/Less than or Equal to and populating a value in this criterion narrows down the list of proformas by the total available funds of the proforma.                               |
| **Owner**              | Selecting values in this criterion narrows down the list of proformas to those owned by the selected timekeeper. More than one timekeeper can be selected.                                           |
| **Forwarded**          | Selecting the check box narrows down the list of proformas that were forwarded to other timekeepers.                                                                                                 |
| **Flagged**            | Selecting the check box narrows down the list of proformas that were marked as priority.                                                                                                             |
| **Approval pending**   | Selecting the check box narrows down the list of proformas that require your approval and have an “Approval pending” sub-status.                                                                     |
