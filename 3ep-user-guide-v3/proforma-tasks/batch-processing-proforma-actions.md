# Batch Processing Proforma Actions

Various proforma actions such as Approve or Bill preview can be performed on a batch of selected proformas.

Do the following to execute an action on multiple proformas:

1. In the Proforma List view, select two or more displayed proformas on which to execute the action.

**Note**: Narrow the list of displayed proformas by clicking on of the Proforma List categories (e.g., In review), typing text (e.g., bill group) in the [**Search**](../getting-started/navigating-3e-proforma-walkthrough.md#navigating-3e-proforma---walkthrough) field, or applying [filters](../getting-started/standard-features-and-navigation/filters.md#filters).

2. Click the **Action** drop-down menu adjacent to the **Select** check box and choose an action (e.g., Bill Preview, Forward, or Submit).
