# About Archived Proformas

The Archived Proformas List view displays all proformas that have been archived in the current and prior periods. Proformas automatically move to the Archived Proformas tab in the following cases:

* When the proforma is invoiced in 3E
* When the proforma is Deferred and the days to archive the proforma have passed. The date when the deferred proforma moves to the Archived Proformas tab is calculated as the proforma generation date plus the number of days to archive the deferred proforma.
* When the proforma status is changed to Closed.

The Archived list displays the 3E Invoice number and Invoice Date associated with the proforma.

Actions available with Archived Proformas include:

* Edit History
* Bill Preview
* Print Proforma
