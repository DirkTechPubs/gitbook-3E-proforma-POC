# Urgent

Selecting the **Urgent** category displays all proformas that are waiting for the user’s review and that have already passed their due date.

&#x20;

Proformas with the following statuses are available:

| **Status** | **Sub-status**   | **Description**                                                                                                  |
| ---------- | ---------------- | ---------------------------------------------------------------------------------------------------------------- |
| **Urgent** | Approval pending | Proformas in this status are waiting for the approver’s review and their urgency date has been met or surpassed. |

\*\* \*\*

The following actions are available:

| **Action**  | **Description**                                                                                                                                                                                                                                                         |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Approve** | Select this action to approve the selected proforma. Approved proformas are sent directly to the Billing department.                                                                                                                                                    |
| **Reject**  | Select this action to reject the selected proforma and send it back to the owner of the proforma. A sub-status in the owner's list indicates that the proforma was rejected. Selecting this action displays a window for entering the reason the proforma was rejected. |
