# Tracking Proforma by Number

![](../../.gitbook/assets/image112.png)

| **Field Definitions**  | **Descriptions**                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Search Criteria**    |                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **Track by**           | Select **Proforma #** and type the number of a proforma to track.                                                                                                                                                                                                                                                                                                                                                                          |
| **Details**            |                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **Proforma Number**    | Displays the proforma number.                                                                                                                                                                                                                                                                                                                                                                                                              |
| **Total**              | Displays the amount due.                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **Current Steps**      | Displays where the proforma is at the workflow process.                                                                                                                                                                                                                                                                                                                                                                                    |
| **Status badge**       | Displays the proforma's current status.                                                                                                                                                                                                                                                                                                                                                                                                    |
| **Owner**              | Displays the proforma's assigned Owner timekeeper.                                                                                                                                                                                                                                                                                                                                                                                         |
| **Client**             | Displays the name and number of the client associated with the proforma.                                                                                                                                                                                                                                                                                                                                                                   |
| **Matter**             | Displays the matter name and number.                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Progress**           | Displays a tracking bar displaying a visual of the proforma's progress through the workflow.                                                                                                                                                                                                                                                                                                                                               |
| **Activity**           | Displays a date stamp of proforma activity (e.g., created, last viewed, etc.)                                                                                                                                                                                                                                                                                                                                                              |
| **Active History Tab** |                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **Modified Date**      | Last date the step was accessed                                                                                                                                                                                                                                                                                                                                                                                                            |
| **Step**               | Name of workflow step                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **Next Action**        | Action triggered when this step was completed.                                                                                                                                                                                                                                                                                                                                                                                             |
| **Name**               | <p>Name of the user assigned to the step. This may show a role when a step is sent to a group of people assigned to a role.</p><p>When the step is accessed, the name of the person who opened the proforma at that step will appear here.</p><p>For example, if an approval is needed, this may initially display 3EProformaApprover. When one of the approvers open the proforma to review it, the step will update with their name.</p> |
| **Role**               | The role of the user as it applies to this proforma.                                                                                                                                                                                                                                                                                                                                                                                       |
| **Activity**           | The status if the workflow step: NotStarted, InProgress, Completed                                                                                                                                                                                                                                                                                                                                                                         |
| **Collaborators**      |                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **Name**               | Collaborator's name                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **Role**               | The role of the user as it applies to this proforma.                                                                                                                                                                                                                                                                                                                                                                                       |
| **Progress**           | <p>Collaborator's status</p><p>Completed = the collaborator has marked the proforma Complete</p><p>In Progress = proforma has been opened but not marked Complete</p><p>Not Started = the collaborator has not yet opened the proforma</p>                                                                                                                                                                                                 |
| **Assignment Date**    | Date the collaborator was added to the proforma                                                                                                                                                                                                                                                                                                                                                                                            |
| **Last Activity**      | Last date the collaborator viewed the proforma                                                                                                                                                                                                                                                                                                                                                                                             |
