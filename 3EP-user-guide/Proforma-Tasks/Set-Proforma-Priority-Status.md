## Set Proforma Priority Status

If the proforma was already set as priority, clicking the flag icon unmarks the proforma as a priority. If the proforma was not a priority, clicking the flag icon marks the proforma as a priority.

Do the following to flag a proforma as a priority.

1.  Locate the proforma in the Proforma list.

2.  Select the **Priority** button for the listed. If the proforma was already set as priority, clicking the flag icon unmarks the proforma as a priority. If the proforma was not a priority, clicking the flag icon marks the proforma as a priority.

**Note**: In Proforma Edit view, click the **Priority** flag in the header section to toggle a proforma's priority status.
