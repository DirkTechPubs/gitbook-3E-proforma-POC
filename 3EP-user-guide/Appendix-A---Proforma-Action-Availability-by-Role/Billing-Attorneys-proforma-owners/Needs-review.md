# Needs review

Selecting the **Needs review** category displays all proformas for which the user is the owner, and which are not yet worked on.

Proformas with the following statuses are available:

| **Status**       | **Sub-status**                                                                                                                                                                                                            | **Description**                                                                                                                          |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **Needs review** | -                                                                                                                                                                                                                         | Proforma is waiting for the billing attorney’s review. Proforma appears on the billing attorney’s dashboard and hasn’t been touched yet. |
| Rejected         | Proformas rejected by approver and returned to the billing attorney’s review.                                                                                                                                             |                                                                                                                                          |
| Returned         | This status is for proformas that were returned to the billing attorney by a biller after they were already submitted and that were opened/edited by the billing attorney.                                                |                                                                                                                                          |
| Biller returned  | This status is for proformas that were returned to the billing attorney by a biller after they were already submitted, and the biller approves/makes specified edits and that were opened/edited by the billing attorney. |                                                                                                                                          |

&#x20;

The following actions are available depending on proforma status/sub-status:

| **Action**         | **Available with status/sub-status**         | **Description**                                                                                                                                                          |
| ------------------ | -------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Print Proforma** | <p>Needs review/all sub-statuses</p><p> </p> | Select this action to view a PDF version of the proforma                                                                                                                 |
| **Bill Preview**   | <p>Needs review/all sub-statuses</p><p> </p> | Select this action to view a PDF version of the bill as it will appear to the client                                                                                     |
| **Close**          | <p>Needs review/all sub-statuses</p><p> </p> | <p>Select this action to close the proforma without saving and remove it from the workflow.</p><p><strong>Note</strong>: Close is only available to Proforma Owners.</p> |
| **Submit**         | <p>Needs review/all sub-statuses</p><p> </p> | Select this action to indicate that all the edits for this proforma have been completed and it should be sent to the biller.                                             |
| **Forward**        | <p>Needs review/all sub-statuses</p><p> </p> | Select this action to forward a proforma to another user.                                                                                                                |
| **Defer**          | <p>Needs review/all sub-statuses</p><p> </p> | Select this action to indicate that work on this proforma should be deferred to a later time                                                                             |
| **Set priority**   | <p>Needs review/all sub-statuses</p><p> </p> | Select this action to set a proforma as priority. Other users who have the proforma in their list will also see that it is marked as priority.                           |
| **Comments**       | <p>Needs review/all sub-statuses</p><p> </p> | Select this action to view/add proforma-level comments.                                                                                                                  |
