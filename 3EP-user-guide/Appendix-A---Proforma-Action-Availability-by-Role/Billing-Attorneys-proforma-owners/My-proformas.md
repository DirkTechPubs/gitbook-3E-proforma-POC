# My proformas

Selecting the My proformas category displays all proformas for which the user is the owner, and which are not yet submitted.

Proformas with the following statuses are available:

| **Status**           | **Sub-status**                                                                                                                                                                                                                                                       | **Description**                                                                                                                                                                                                     |
| -------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Needs review**     | -                                                                                                                                                                                                                                                                    | New proformas that have not been opened or edited.                                                                                                                                                                  |
| Rejected             | Proformas rejected by an approver and returned to the billing attorney for further review.                                                                                                                                                                           |                                                                                                                                                                                                                     |
| Returned             | This status is for proformas that were returned to the billing attorney by a biller after it was already submitted and that were opened/edited by the billing attorney.                                                                                              |                                                                                                                                                                                                                     |
| Biller returned      | This status is for proformas that were returned to the billing attorney by a biller after it was already submitted and the biller approves/makes specified edits and that were opened/edited by the billing attorney.                                                |                                                                                                                                                                                                                     |
| **In review**        | -                                                                                                                                                                                                                                                                    | Proformas opened and/or edited by the user.                                                                                                                                                                         |
| Rejected             | Proformas rejected by an approver and returned to the billing attorney and were opened/edited by the billing attorney.                                                                                                                                               |                                                                                                                                                                                                                     |
| Returned             | This status is for proformas that were returned to the billing attorney by a biller after they were already submitted and that were opened/edited by the billing attorney.                                                                                           |                                                                                                                                                                                                                     |
| Biller returned      | This status is for proformas that were returned to the billing attorney by a biller after it was already submitted and the biller approves/makes specified edits and that were opened/edited by the billing attorney.                                                |                                                                                                                                                                                                                     |
| **Urgent**           | -                                                                                                                                                                                                                                                                    | <p>Proformas in this list have met or surpassed the calculated “due date”.</p><p><strong>Note:</strong>Urgency date is calculated based on a setting in 3E Override / Set System Options and the proforma date.</p> |
| Rejected             | Proformas rejected by an approver and returned to the billing attorney and the urgency date has been met or surpassed.                                                                                                                                               |                                                                                                                                                                                                                     |
| Returned             | This status is for proformas that were returned to the billing attorney by a biller after it was already submitted and that were opened/edited by the billing attorney.                                                                                              |                                                                                                                                                                                                                     |
| Biller returned      | This status is for proformas that were returned to the billing attorney by a biller after it was already submitted, and the biller approves/makes specified edits and that were opened/edited by the billing attorney. Their urgency date has been met or surpassed. |                                                                                                                                                                                                                     |
| **Approval pending** | -                                                                                                                                                                                                                                                                    | Proforma is waiting to be reviewed by an approver                                                                                                                                                                   |

&#x20;

The following actions are available depending on proforma status/sub-status:

| **Action**         | **Available with status/sub-status**                                                                                                                        | **Description**                                                                                                                                                          |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Print Proforma** | <p>Needs review/all sub-statuses</p><p>In review/all sub-statuses</p><p>Urgent/all sub-statuses</p><p>Approval pending</p>                                  | Select this action to view a PDF version of the proforma                                                                                                                 |
| **Bill Preview**   | <p>Needs review/all sub-statuses</p><p>In review/all sub-statuses</p><p>Urgent/all sub-statuses</p><p>Approval pending</p>                                  | Select this action to view a PDF version of the bill as it will appear to the client                                                                                     |
| **Close**          | <p>Needs review/all sub-statuses</p><p>In review/all sub-statuses</p><p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p>                              | <p>Select this action to close the proforma without saving and remove it from the workflow.</p><p><strong>Note</strong>: Close is only available to Proforma Owners.</p> |
| **Submit**         | <p>Needs review/all sub-statuses</p><p>In review/all sub-statuses</p><p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p><p>Urgent/Biller returned</p> | Select this action to indicate that all the edits for this proforma have been completed and it should be sent to the biller.                                             |
| **Reclaim**        | <p>Approval pending</p><p>Urgent/Completed</p><p>Completed</p><p>Completed/Approved</p><p>Completed/Biller review</p>                                       | Select this action to return a proforma back to editing after it was submitted                                                                                           |
| **Forward**        | <p>Needs review/all sub-statuses</p><p>In review/all sub-statuses</p><p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p><p>Urgent/Biller returned</p> | Select this action to forward a proforma to another user.                                                                                                                |
| **Defer**          | <p>Needs review/all sub-statuses</p><p>In review/all sub-statuses</p><p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p><p>Urgent/Biller returned</p> | Select this action to indicate that work on this proforma should be deferred to a later time                                                                             |
| **Set priority**   | <p>Needs review/all sub-statuses</p><p>In review/all sub-statuses</p><p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p><p>Urgent/Biller returned</p> | Select this action to set a proforma as priority. Other users who have the proforma in their list will also see that it is marked as priority.                           |
| **Comments**       | <p>Needs review/all sub-statuses</p><p>In review/all sub-statuses</p><p>Urgent</p><p>Urgent/Returned</p><p>Urgent/Rejected</p><p>Urgent/Biller returned</p> | Select this action to view/add proforma-level comments.                                                                                                                  |
