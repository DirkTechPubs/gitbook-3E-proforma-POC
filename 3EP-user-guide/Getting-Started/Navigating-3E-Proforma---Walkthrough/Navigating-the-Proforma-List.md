# Navigating the Proforma List

3E Proforma provides multiple options to enable you to quickly find the proformas you need to manage. You can use the following options to control what displays in the Proforma List View:

* [Categories](Navigating-the-Proforma-List.md#categories)
* [Action Menus](Navigating-the-Proforma-List.md#action-menus)
* [Filters](Navigating-the-Proforma-List.md#filters)
* [Search](Navigating-the-Proforma-List.md#search)
* [Sort](Navigating-the-Proforma-List.md#sort)
* [Paging](Navigating-the-Proforma-List.md#paging)

## Categories

The proformas that display in the list depend on which Proforma category is selected. The actions that can be performed on a proforma also change.

The actions that can be performed in the Proformas List depend on the role of the current user for the exact proforma (Timekeeper, Billing Attorney or Approver).

Some proformas in a category have both a status and a sub-status.

## Action Menus

Tasks such as forwarding, marking complete, or printing can be can be initiated by making a selection from one of the Action menus on the Proforma List page. There are two types of action menus available on the Proforma List.

* **Batch Action Menu** - Use to apply actions (e.g., Mark Complete or Defer) to multiple proformas simultaneously. See [Batch Processing Proforma Actions](../../Proforma-Tasks/Batch-Processing-Proforma-Actions.md#batch-processing-proforma-actions) for further details.
* **Proforma Level Action Menu** - Use to apply an action for an individual proforma displayed in the list.

**Note**: The availability of actions (e.g., Add Costs, Add Fees, etc.) is determined by 3E User/Role security. See [Appendix A - Proforma Action Availability by Role](../../Appendix-A---Proforma-Action-Availability-by-Role.md#appendix-a---proforma-action-availability-by-role) for additional details.

## Filters

You can apply filter criteria to narrow the list of proformas that display in the list. You can apply filter criteria such as client name, matter number, fee values, and more. See [Filters](../Standard-Features-and-Navigation/Filters.md#filters) for additional details.

## Search

You can conduct a quick search for a proforma by entering specific search criteria (e.g., client name, proforma number, or bill group).

## Sort

You can sort the list by criteria such as client name, fees, and costs. Additionally, the list can be ordered, based on the sorting criteria, from A - Z or Z to A.

## Paging

A paging bar at the bottom of the list enables you to move forwards or backwards in a multi-page list of proformas. You specify a specific page to jump to and control how many proformas display per page. On long pages, you have the option to quickly jump back to the top of the page.
