# Side Navigation Menu

The navigation bar on the left side of the screen contains the different views and menus that are available in 3E Proforma.

![](../../.gitbook/assets/image9.png)

**Toggle Navigation Menu View**

Click on the **Toggle Navigation Menu** icon ![](../../.gitbook/assets/image10.png) to expand the toolbar to a menu with labels.

**Proforma Lists** ![](../../.gitbook/assets/image11.png)

Click to go to the Proforma List view.

**WIP List** ![](../../.gitbook/assets/image12.png)

Click to view a list of Matters for which you are the assigned Billing Attorney, or you are working as the assigned Billing Attorney

**Note**: The WIP list view displays for the Billing Attorney only when the organization has enabled this feature.

**Archived Proformas** ![](../../.gitbook/assets/image13.png)

Click to view displays all proformas that have been archived in the current and prior periods.

**Proforma Tracking**![](../../.gitbook/assets/image14.png)

Click to view a proforma's progress through its workflow.

**Settings** ![](../../.gitbook/assets/image15.png)

The Settings menu contains user configuration settings, which allow a user to set default behavior for certain areas of the system.&#x20;

**Help** ![](../../.gitbook/assets/image16.png)

Click to access 3E Proforma's online help.

**About** ![](../../.gitbook/assets/image17.png)

Click to display the 3E Proformas' version information as well as a link to the 3E instance URL and the Elite privacy statement.

**Go to Workspace** ![](../../.gitbook/assets/image18.png)

Click to open the Workspace application.

**Sign out** ![](../../.gitbook/assets/image19.png)

Click to log out of 3E Proforma.
