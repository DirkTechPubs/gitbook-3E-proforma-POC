### **Workflow Config specific settings under 3E Proforma group**

| **Option** | **Description** | **Default Setting** |
|----|----|----|
| **Approve Bill Description** | Description for the Workflow Approve Bill step | Approve Bill |
| **Approve Bill Process** | Process id for the Approve Bill step | WF_ProformaApprove |
| **Approver Role** | Name of the Role to assign users as approvers | 3EProformaApproverRole |
| **Biller Edit Bill Description** | Description for the Workflow Biller Edit Bill step | Biller Edit Bill |
| **Edit Bill Description** | Description for the Workflow Edit Bill step | Edit Bill |
| **Edit Bill Process** | Process id for the Workflow Edit Bill step | WF_ProformaEdit |
| **Generate Proforma Process** | Process id for the Generate Proforma step | WF_ProfGeneration |
| **Group Proforma Workflow Name** | Group Proforma Workflow Name | WF_ProfGroup |
| **Process Bill** | Process ID for the Process Bill step | WF_ProformaBill |
| **Proforma Workflow Name** | Billing Journey Workflow Name | 3EProformaBilling |
| **Start Group Workflow Process Name ** | Process ID to start Group Billing Workflow | WF_ProfGroupStart |
| **Start Workflow from Group Return Process Name** | Process ID to start the Billing Workflow when returning from Group Bill | WF_ProfStartFromGroup |
| **Start Workflow Process Name** | Process ID to start the Billing workflow | WF_ProfStart |
| **WF Bill Write Down Approved Rule Set Code** | The code assigned to the 'Bill write down approved' ruleset under 'Approve Bill' | ProfApproveDef |
| **WF Biller Return Rule Set Code** | The code assigned to the 'Biller Return' ruleset under 'Return Biller Proforma' in the workflow | BillerEditProfReturnDef |
| **WF Group Bill Submit Rule Set Code** | The code assigned to the 'Group Bill' ruleset without Approval | GroupProfSubmitDef |
| **WF Group Bill Write Down Approved Rule Set Code** | The code assigned to the 'Group Bill Write Down Approved' ruleset under 'Approve Bill' in the workflow | GroupProfApproveDef |
| **WF Reject Proforma Rule Set Code** | The code assigned to the 'Reject Proforma' ruleset under 'Approve Bill' in the workflow | ProfRejectDef |
| **WF Return Group Proforma Rule Set Code** | The code assigned to the 'Return Group Proforma' ruleset under 'Process Group Bill' in the workflow | GroupProfBillReturnDef |
| **WF Return Proforma Rule Set Code** | The code assigned to the 'Return Proforma' ruleset under 'Process Bill' in the workflow | ProfBillReturnDef |
