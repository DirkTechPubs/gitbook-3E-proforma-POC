## Archived Proformas Form and Field Definitions

The Archived Proformas List view displays all proformas that have been archived in the current and prior periods.

<img src="../3EP-UG-img/media/image125.png" style="width:6.5in;height:3.60417in" />

 

| **Field Name** | **Description** |
|----|----|
| **Select** | Select this check box to flag a proforma for an action (e.g., Bill Preview or Edit History) |
| **Invoice number** | Displays the invoice number. |
| **Invoice date** | Displays the date the invoice was created. |
| **Proforma number** | Displays the proforma number. Click this number to drill down to Proforma Detail view. |
| **Client** | Displays the name of the client associated with the matter. |
| **Matter** | Displays the matter name. |
| **Bill Group** | Displays the bill group to which the group |
| **Forwarded** | Indicates the proforma was forwarded to another timekeeper. |
| **Total** | Displays the total amount billed. |
| **Total fees** | Displays the total amount of fees. |
