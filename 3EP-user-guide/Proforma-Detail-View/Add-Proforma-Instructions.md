# Add Proforma Instructions

Internal proforma instructions can be added for a Biller to follow.

Do the following to add instructions:

1. Locate the proforma in the Proforma List and click the proforma number to access the proforma details.
2. In the header section, click the **Expand** icon ![](../.gitbook/assets/image89.png) to display the proforma totals section.
3. Enter text in the **Instructions** field.
4. Click **Save & Close**.
