# Add Adjustment Form and Field Definitions

The options on this form are used to create an adjustment entry.

![](../../.gitbook/assets/image100.png)

&#x20;

| **Field Name**               | **Description**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Type**                     | Select an Adjustment Type from the drop down list.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **Method**                   | Select an option from this drop-down list to determine how the adjustment is applied. The adjustment method will default to “Proportional by entries on the proforma”. There is a set list of adjustment methods to choose from and the method can be modified from what is defaulted.                                                                                                                                                                                                                                                                                                                                                                                                          |
| **Start date**               | Select a start date for which time cards will be considered for the adjustment. Only time cards with a date that is on or after the date entered are selected.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **End date**                 | Select an end date for which time cards will be considered for the adjustment. Only time cards with a date that is on or before the date entered are selected.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Include Flat Fee Charges** | Select this check box to include the flat fee charges in the adjustment if the matter has a flat fee agreement.  This field is only available when there is a flat fee charge on the proforma and the _Adjust Bill_ option is set to **Percentage**.                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **Timekeeper**               | Select the timekeeper/fee earner whose fees should be adjusted.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **Current Total**            | Displays the current proforma total. If you have made other edits to cards that would affect the proforma total but have not clicked on Save & recalc, you may not see the edited value as the current total.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **Adjust Bill**              | <p>Choose one of the following:</p><ul><li><strong>Net bill amount</strong> – Type the total of what the actual bill amount should be. It will automatically calculate the amount of the adjustment in the Amount field.</li><li><strong>Amount</strong> – Type the amount of the adjustment. To adjust the bill amount down, enter a minus sign (-) before the amount of the desired adjustment.</li><li><strong>Percentage</strong> – Type the percentage of the adjustment. To apply a percentage adjustment that adjusts the bill total down, enter a minus sign (-) before the percent discount desired. For example, -10 in the percentage field will decrease the fees by 10%.</li></ul> |
| **Adjusted Total**           | Displays a value based on the amount or percentage entered. It allows you to view how the adjustment impacts the overall Proforma total prior to clicking on **Save adjustment**.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| **Cancel**                   | Click to exit this form                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Save adjustment**          | <p>Click to save your changes and close this form</p><p><strong>Note</strong>: This button is inactive until the required fields and an adjustment amount or percentage is entered.</p>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
