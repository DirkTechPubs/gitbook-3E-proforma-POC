# Proforma Details - Charges Tab

Click the **Charges** tab to display any charges on the proforma, such as calculated interest or tax.

**Note:** Not all charges are actionable or editable. For example, calculated charges such as tax or interest are not editable. Some charges, such as tax, are not actionable.

&#x20;

![](../../.gitbook/assets/image81.png)

The table below describes the attributes of the charge card:

| **Attribute**           | **Description**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Timekeeper              | This attribute displays the name of the timekeeper, timekeeper number, and the title of the timekeeper.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Date**                | This attribute shows the date of the charge.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **Charge type**         | This attribute shows the charge type of the charge card.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **Presentation Amount** | <p>This field displays the amount of the flat fee charge(s) on flat fee proformas.</p><p><strong>Note</strong>: The Amount field displays the <em>FlatFee Amount</em> - <em>Total Fee Amount</em>.</p><p>For example, if the flat fee charge should be $300 and there are $800 of fees on the proforma, the Presentation Amount on the charge will display $300 and the Amount will display ($500), which is the amount of the flat fee less the total fees with that time type.</p>                                                                                                                                                                                     |
| **Amount**              | The Amount attribute displays the worked amount for the associated timekeeper for this charge entry. To change the amount to be billed, click in the Amount input field and enter the new amount. To update proforma totals based on edits, click the **Save & recalc** button on top of the proforma.                                                                                                                                                                                                                                                                                                                                                                   |
| **Narrative**           | Displays narrative text that is displayed on the bill. Click **Read more/Edit** to display the full text or make changes if using Card View. When using Grid View, click the Narrative field and click the **Expand** icon ![](../../.gitbook/assets/image23.png). See [Card Narrative Fields](../../Getting-Started/Standard-Features-and-Navigation/Card-Narrative-Fields.md#card-narrative-fields) for further details.                                                                                                                                                                                                                                               |
| **Comments**            | <p>Where desired or if required based on a card action, populate the Comments field with the comment. When the comment field is populated, it becomes read-only with a “Read more/Edit” link. Click the <strong>Read more/Edit</strong> link to edit the comment field in the pop-up.</p><p>Information entered into the Comment (bottom) field is for the billing staff and is not displayed on the actual bill.</p><p> </p>                                                                                                                                                                                                                                            |
| **Code Fields**         | <p>To browse codes, expand the card by clicking the Expand icon <img src="../../.gitbook/assets/image78.png" alt="">.</p><p>These fields display the codes associated with the time card, such as: Phase Codes, Task Codes, Activity Codes, and Tax codes.  Phase, Task, and Activity fields display the Code (above) and the Description (below the Code).</p><p><img src="../../.gitbook/assets/image79.png" alt="" data-size="original"></p><p><strong>Note:</strong>  If you wish to see the Code fields always displayed, open Settings <img src="../../.gitbook/assets/image15.png" alt="">, and set <strong>Automatically expand code fields</strong> to Yes.</p> |
