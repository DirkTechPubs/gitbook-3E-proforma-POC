# Proforma Details – Adjustments Tab

The Adjustment tab is used to add Adjustment cards used to adjust the amount of the entire proforma total by an amount or a percent. Adjustment cards can also be edited and deleted.

![](../../.gitbook/assets/image82.png)

| **Field Name**     | **Description**                                                                                                                                                                                 |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Add adjustment** | Click to create an adjustment for the proforma.                                                                                                                                                 |
| **Timekeeper**     | Displays the name of the timekeeper whose fees have been adjusted.                                                                                                                              |
| **Start date**     | Displays the start date for which time cards will be considered for the adjustment.                                                                                                             |
| **End date**       | Displays the end date for which time cards will be considered for the adjustment.                                                                                                               |
| **Type**           | Displays the apply adjustment type.                                                                                                                                                             |
| **Method**         | Displays the how the adjustment is applied.                                                                                                                                                     |
| **Amount**         | Displays the adjusted total.                                                                                                                                                                    |
| **Action Menu**    | <p>Select one of the adjustment actions as needed.</p><ul><li><strong>Edit</strong> - Select to edit an adjustment.</li><li><strong>Delete</strong> - Select to delete an adjustment.</li></ul> |
