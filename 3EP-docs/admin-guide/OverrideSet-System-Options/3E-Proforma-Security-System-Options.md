## 3E Proforma Security System Options

Beginning with 3E 5.8/3E on prem 3.2, there is a **3E Proforma Security group** in the **3E Override/Set System Options**.

Administrators will use these settings to easily show or hide fields, or mark them as read-only, ensuring the interface aligns precisely with the firm’s workflows and compliance needs. Best of all, these settings can be applied at the **firm, unit, role, or individual user level** – giving unmatched flexibility to adapt the application to each unique business structure.

Previously, some UI elements were managed through User/Role settings. With the introduction of this system options group, everything is centralized—making configuration simpler and more powerful.

See *3E Proforma - System Options Configuration Worksheet* (KC No. 000035886) for a configuration planning worksheet to assist with setup and testing of these settings.
