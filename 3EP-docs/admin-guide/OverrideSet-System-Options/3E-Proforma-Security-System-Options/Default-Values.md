### **Default Values**

The system default values are set to recreate the previous stock functionality of 3E Proforma – if something was visible and editable before, then the system default will also have that behavior. All options are set to visible/full access except for **Show_WIPList**, which was previously hidden by default.

Note that **Show_WIPList** was previously in the 3E Proforma system options group and will need to be set up to display if needed in the 3E Proforma Security group.
