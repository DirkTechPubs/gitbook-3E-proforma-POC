## About 3E Proforma

3E Proforma (3EP) is the only cloud-native proforma and billing solution designed for law firms that require a comprehensive, integrated, and secure platform to manage their billing processes. 3E Proforma is integrated directly within the 3E ecosystem which allows for real-time data synchronization, seamless collaboration, and more flexibility in managing workflows. 3E Proforma is not standalone and requires 3E Cloud to operate.
