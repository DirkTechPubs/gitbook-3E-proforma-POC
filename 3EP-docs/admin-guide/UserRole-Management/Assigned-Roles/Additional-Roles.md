### **Additional Roles**

- **3EProformaApproverRole** – used for workflow routing and controls the Approve/Reject actions. If the user will be an approver for write downs, assign this role to them.

- **3EProformaInvGeneratorRole** – Used for workflow routing and controls the ability to generate invoices in 3E Proforma. See [<u>WIP List and Generate Proforma(s) – Optional 3E Proforma functionality</u>](../../WIP-List-and-Generate-Proformas--Optional-3E-Proforma-functionality.md#wip-list-and-generate-proformas--optional-3e-proforma-functionality) for more information.

- **3EProformaAdminRole** – Controls the ability to view firm-wide settings in 3E Proforma.

- **3E ProformaWriteOffRole** – For on premises customers on a 3E version below 3.2, this role is used to allow the Write Off Fees and Write Off Costs checkboxes in Proforma Details.

Cloud customers and on premises customers on 3E 3.2 or higher no longer use this role, instead access to those checkboxes is controlled by the **ProfDetails_Details_GroupEdit_WriteOffCosts** and **ProfDetails_Details_GroupEdit_WriteOffFees** system options.
