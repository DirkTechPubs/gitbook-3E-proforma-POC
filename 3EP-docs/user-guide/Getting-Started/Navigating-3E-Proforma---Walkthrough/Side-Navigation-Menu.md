### **Side Navigation Menu**

The navigation bar on the left side of the screen contains the different views and menus that are available in 3E Proforma.

<img src="../../3EP-UG-img/media/image9.png" style="width:6.5in;height:7.89583in" />

**Toggle Navigation Menu View**

Click on the **Toggle Navigation Menu** icon <img src="../../3EP-UG-img/media/image10.png" style="width:0.23958in;height:0.21875in" /> to expand the toolbar to a menu with labels.

**Proforma Lists** <img src="../../3EP-UG-img/media/image11.png" style="width:0.17708in;height:0.1875in" />

Click to go to the Proforma List view.

**WIP List** <img src="../../3EP-UG-img/media/image12.png" style="width:0.22917in;height:0.20833in" />

Click to view a list of Matters for which you are the assigned Billing Attorney, or you are working as the assigned Billing Attorney

**Note**: The WIP list view displays for the Billing Attorney only when the organization has enabled this feature.

**Archived Proformas** <img src="../../3EP-UG-img/media/image13.png" style="width:0.17708in;height:0.1875in" />

Click to view displays all proformas that have been archived in the current and prior periods.

**Proforma Tracking**<img src="../../3EP-UG-img/media/image14.png" style="width:0.21875in;height:0.23958in" />

Click to view a proforma's progress through its workflow.

**Settings** <img src="../../3EP-UG-img/media/image15.png" style="width:0.1875in;height:0.19792in" />

The Settings menu contains user configuration settings, which allow a user to set default behavior for certain areas of the system. 

**Help** <img src="../../3EP-UG-img/media/image16.png" style="width:0.17708in;height:0.17708in" />

Click to access 3E Proforma's online help.

**About** <img src="../../3EP-UG-img/media/image17.png" style="width:0.19792in;height:0.17708in" />

Click to display the 3E Proformas' version information as well as a link to the 3E instance URL and the Elite privacy statement.

**Go to Workspace** <img src="../../3EP-UG-img/media/image18.png" style="width:0.21875in;height:0.22917in" />

Click to open the Workspace application.

**Sign out** <img src="../../3EP-UG-img/media/image19.png" style="width:0.20833in;height:0.19792in" />

Click to log out of 3E Proforma.
