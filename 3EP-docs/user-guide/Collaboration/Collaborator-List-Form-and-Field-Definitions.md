## Collaborator List Form and Field Definitions

The Collaborator list displays a list of timekeepers working on a proforma.

<img src="../3EP-UG-img/media/image118.png" style="width:6.5in;height:2.30208in" />

The table below describes the columns on the Collaboration Details section.

| **Field Name** | **Description** |
|----|----|
| **Add collaborator** | Click to add a collaborator to the list. |
| **Remove collaborators** | Click to remove multiple collaborators from the list simultaneously. This action only removes collaborators with edit rights. The Owner and Co-Owner will not be affected. Click **Save** to save the changes or **Cancel** to revert the collaborators back to the original list. |
| **Clear** | Click to undo additions to the list before saving. |
| **Collaborator Grid** | ** ** |
| **Name** | This column displays the name of all timekeepers who are part of the collaboration group for this proforma. |
| **Title** | This column displays the title of the timekeeper. |
| **Office** | This column displays the location of the timekeeper. |
| **ID** | This column displays the timekeeper’s unique identification number. |
| **Open** | This column displays a check mark if a timekeeper has the proforma currently open. |
| Complete | This column displays a check mark if the timekeeper has marked the proforma as complete. |
| **Assignment date** | This column displays the date that the rights were assigned to the timekeeper. |
| **Rights** | This column displays the rights assigned for each timekeeper for the proforma. The Owner ad [<u>Co-owner</u>](Assign-Co-owner-to-Proforma.md#assign-co-owner-to-proforma) are identified in this column. |
| **Delete** <img src="../3EP-UG-img/media/image113.png" style="width:0.13542in;height:0.13542in" /> | Click to remove a collaborator from the list. |
