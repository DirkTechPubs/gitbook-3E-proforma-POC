### **Reject Proforma**

Do the following to reject a proforma

1.  Click on the number of a proforma displayed in the Proforma list to access the proforma details.

2.  Review the changes that have been made to the proforma.

3.  Select **Reject** from the **Proforma Action** menu. A message displays that "Proforma XXXX has been successfully returned." The proforma is removed from the list of Needs review and is returned to the Billing Attorney.
