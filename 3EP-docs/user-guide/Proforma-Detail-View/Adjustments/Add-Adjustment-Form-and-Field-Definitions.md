### **Add Adjustment Form and Field Definitions**

The options on this form are used to create an adjustment entry.

<img src="../../3EP-UG-img/media/image100.png" style="width:6.5in;height:4.66667in" />

 

<table style="width:96%;">
<colgroup>
<col style="width: 30%" />
<col style="width: 65%" />
</colgroup>
<thead>
<tr>
<th><strong>Field Name</strong></th>
<th><strong>Description</strong></th>
</tr>
</thead>
<tbody>
<tr>
<td><strong>Type</strong></td>
<td>Select an Adjustment Type from the drop down list.</td>
</tr>
<tr>
<td><strong>Method</strong></td>
<td>Select an option from this drop-down list to determine how the adjustment is applied. The adjustment method will default to “Proportional by entries on the proforma”. There is a set list of adjustment methods to choose from and the method can be modified from what is defaulted.</td>
</tr>
<tr>
<td><strong>Start date</strong></td>
<td>Select a start date for which time cards will be considered for the adjustment. Only time cards with a date that is on or after the date entered are selected.</td>
</tr>
<tr>
<td><strong>End date</strong></td>
<td>Select an end date for which time cards will be considered for the adjustment. Only time cards with a date that is on or before the date entered are selected.</td>
</tr>
<tr>
<td><strong>Include Flat Fee Charges</strong></td>
<td>Select this check box to include the flat fee charges in the adjustment if the matter has a flat fee agreement.  This field is only available when there is a flat fee charge on the proforma and the <em>Adjust Bill</em> option is set to <strong>Percentage</strong>.</td>
</tr>
<tr>
<td><strong>Timekeeper</strong></td>
<td>Select the timekeeper/fee earner whose fees should be adjusted.</td>
</tr>
<tr>
<td><strong>Current Total</strong></td>
<td>Displays the current proforma total. If you have made other edits to cards that would affect the proforma total but have not clicked on Save &amp; recalc, you may not see the edited value as the current total.</td>
</tr>
<tr>
<td><strong>Adjust Bill</strong></td>
<td><p>Choose one of the following:</p>
<ul>
<li><p><strong>Net bill amount</strong> – Type the total of what the actual bill amount should be. It will automatically calculate the amount of the adjustment in the Amount field.</p></li>
<li><p><strong>Amount</strong> – Type the amount of the adjustment. To adjust the bill amount down, enter a minus sign (-) before the amount of the desired adjustment.</p></li>
<li><p><strong>Percentage</strong> – Type the percentage of the adjustment. To apply a percentage adjustment that adjusts the bill total down, enter a minus sign (-) before the percent discount desired. For example, -10 in the percentage field will decrease the fees by 10%.</p></li>
</ul></td>
</tr>
<tr>
<td><strong>Adjusted Total</strong></td>
<td>Displays a value based on the amount or percentage entered. It allows you to view how the adjustment impacts the overall Proforma total prior to clicking on <strong>Save adjustment</strong>.</td>
</tr>
<tr>
<td><strong>Cancel</strong></td>
<td>Click to exit this form</td>
</tr>
<tr>
<td><strong>Save adjustment</strong></td>
<td><p>Click to save your changes and close this form</p>
<p><strong>Note</strong>: This button is inactive until the required fields and an adjustment amount or percentage is entered.</p></td>
</tr>
</tbody>
</table>
