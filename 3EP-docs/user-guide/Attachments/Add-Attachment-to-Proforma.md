## Add Attachment to Proforma

Do the following to attach a file to a proforma:

1.  Access the [<u>Proforma Details</u>](../Proforma-Detail-View.md#proforma-detail-view) for a proforma.

2.  Click the **Attachment** button.

3.  Click **Upload File** to add an attachment.

4.  Browse to and select a document to attach.

5.  Click **Open** to attach the file. Repeat steps 3-5 to add additional attachments.

6.  Click **Close** when done.
