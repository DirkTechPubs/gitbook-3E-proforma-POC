## Working As Another User

Do the following to work as a different timekeeper:

1.  On the [<u>Global Toolbar</u>](navigating-3e-proforma-walkthrough/global-toolbar.md#global-toolbar), you will see your Timekeeper’s name. Click the **Down** arrow next to your name to see other timekeepers that have delegated access to their proformas.  If the list is long, you can use the search field to find the timekeeper instead of scrolling.

2.  To view the proformas available to another timekeeper, select the timekeeper from the drop-down list.

3.  Click **Switch timekeepers** on the confirmation message. Your views will change according to what proformas are associated with the selected timekeeper.

For example, if the logged-in timekeeper (Kathryn Jones) selects Jean Foster from the list, then Kathryn Jones will have access to the proformas that are available to Jean Foster.

**Note**: Delegation is set up in the **User Timekeeper Map** form in 3E. For more information about setting up delegation in the User Timekeeper Map, see the [*User/Role Management*](https://help.elite.com/3e-help-mlf/3Ehelp.htm#Subsystems/Elite 3E Administration Guide/About_User_Role_Management.htm) section in the 3E Administration Guide.
