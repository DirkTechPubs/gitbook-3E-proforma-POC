## Exiting Proforma Detail View

There are multiple ways to exit the Proforma Detail view. The following covers the differences between the main three actions to exit this page.

**Note**: Certain workflow actions such as Mark as Complete, Approve, or Submit will also close the Proforma Detail view and return a user to the Proforma List. The following information only applies to non-workflow actions used to exit this view.

- **Save & Close** - The Save & close button saves all edits made to the proforma, recalculates proforma totals and sends the user back to the list of proformas from which he opened the proforma details. The proforma is unlocked and becomes available to other users.

- **Close** - The owner clicks Close to exit the proforma without saving any changes and remove it from the workflow.

- **Cancel** - Click Cancel to unlock the proforma and cancel all unsaved changes. The proforma details page is closed, and the user is back to the proforma list from which proforma details were opened.
