# Proforma Details – Payors Tab

The Payors tab displays information about the matter's payor(s), including multi-payor details, and Payor by layer information.

**Note**: The Payors tab only displays when the 3E system option [**Allow WF\_Prof\_Edit\_Payor**](../../overrides-and-system-options/3e-proforma-security-system-options-list.md) is set to **View**. Additionally, information displayed in the Payors grid is read-only.

![](../../.gitbook/assets/image86.png)

| **Field Name**            | **Descriptions**                                                                                                                                      |
| ------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Payor Number**          | Displays the payor's number.                                                                                                                          |
| **Payor Name**            | Displays the name of the payor responsible for a portion of the proforma.                                                                             |
| **Client Number**         | Displays the number of the client assigned to this payor.                                                                                             |
| **Client Name**           | Displays the client assigned to this payor.                                                                                                           |
| **Site Number**           | Displays the number of the payor's billing site.                                                                                                      |
| **Site Name**             | Displays the name of the billing site for this payor.                                                                                                 |
| **Statement Site Number** | Displays the number associated with the statement site.                                                                                               |
| **Statement Site Name**   | Displays the site where the matter multi payor’s consolidated statements will be mailed.                                                              |
| **For Attention Of**      | Displays the payor's contact name. Used as part of 3E's multipayor layors functionality.                                                              |
| **Reference Number**      | Displays the client's reference for the firm's matter by payor and layer.                                                                             |
| **Fee Percent**           | Displays the percentage of fees that the payor will be responsible for paying.                                                                        |
| **Hard Cost Percent**     | Displays the percentage of hard costs that the payor will be responsible for paying.                                                                  |
| **Soft Cost Percent**     | Displays the percentage of soft costs that the payor will be responsible for paying.                                                                  |
| **Tax Percent**           | Displays the percentage of taxes that the payor will be responsible for paying.                                                                       |
| **Interest Percent**      | Displays the percentage of interest that the payor will be responsible for paying.                                                                    |
| **BOA Percent**           | Displays the percentage of bill on account that this payor will be responsible for paying.                                                            |
| **Other Percent**         | Displays the percentage of other that this payor will be responsible for paying.                                                                      |
| **Amount Fees**           | Displays the amount of fees that this payor will be responsible for paying. Any amounts due to rounding are assigned to the Default Payor.            |
| **Amount Hard Costs**     | Displays the amount of hard costs that this payor will be responsible for paying. Any amounts due to rounding are assigned to the Default Payor.      |
| **Amount Soft Costs**     | Displays the amount of soft costs that this payor will be responsible for paying. Any amounts due to rounding are assigned to the Default Payor.      |
| **Amount Taxes**          | Displays the amount of soft costs that this payor will be responsible for paying. Any amounts due to rounding are assigned to the Default Payor.      |
| **Amount Interest**       | Displays the amount of interest that this payor will be responsible for paying. Any amounts due to rounding are assigned to the Default Payor.        |
| **Amount BOA**            | Displays the amount of bill on account that this payor will be responsible for paying. Any amounts due to rounding are assigned to the Default Payor. |
| **Amount Other**          | Displays the amount of other that this payor will be responsible for paying. Any amounts due to rounding are assigned to the Default Payor.           |
