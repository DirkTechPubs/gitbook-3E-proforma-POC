## Add Note to Proforma

Do the following to add a note to a proforma:

1.  Access the [<u>Proforma Details</u>](../proforma-detail-view.md#proforma-detail-view) for a proforma.

2.  Click the **Attachment** button.

3.  Click **Add Note**.

4.  Edit the default note title (i.e., Note 1).

5.  Type text for the note in the following field.

6.  Click **Save Note**. The name of the note will appear in the attachments list.
