## Browse Archived Proformas

Do the following to view a list of archived proformas:

1.  Select **Archived Proformas** in the side Navigation menu.

<!-- -->

2.  Specify filtering criteria in the Filter panel to narrow the list of displayed proformas and click **Apply**.

3.  Do one of the following:

- **Batch Action** - You can generate a **Bill Preview** or **Print Proforma** for a batch of selected archived proforma.

- **Single Action** - Click the proforma number for a listed proforma to access the Proforma Details view for an archived proforma. Select actions such as **Edit History**, **Bill Preview**, or **Print Proforma** from the Proforma-level **Action** menu.
