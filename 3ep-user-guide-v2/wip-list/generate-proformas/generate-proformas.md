# Generate Proformas

Do the following to generate proformas for matters in the WIP list:

1. Click **WIP List** on the main [**Navigation Menu**](../../getting-started/navigating-3e-proforma-walkthrough/side-navigation-menu.md#side-navigation-menu).
2. Click the **Filter** icon to narrow the list of matters displayed.
3. Configure settings such as the date range as needed. See the [WIP List Form and Field Definitions](../wip-list-form-and-field-definitions.md#wip-list-form-and-field-definitions) for details.
4. Do one of the following:

* **Single Proforma** - To generate a proforma for a single or group Matter, click the **Generate** button directly in the row for the Matter for which you want to generate a proforma.
* **Multiple Proformas** - To generate proformas for all or multiple Matters in the WIP list, click the **Select all check** box or select the check boxes for multiple matters in the list. The select the **Actions > Generate** action.

5. Set configuration for generating proformas on the [Proforma Generation](proforma-generation-form-and-field-definitions.md#proforma-generation-form-and-field-definitions) form.

**Note**: A Proforma Generation pop-up displays, with settings defined by your organization according to the desired outcome when generating proformas. The pop-up is set up specifically for your Timekeeper ID, so any changes you make in the pop-up will only affect you and ONLY for the specific proforma generation you are able to perform. The next time this pop-up displays it will contain the default settings. If you want your default settings changed, please consult with your organization’s administrator.

6. Click the Review tab to review the proformas selected for generation.
7. Click **Generate**.
