# Icon Legend

The following is a list of the most common icons used to access functions in 3E Proforma (e.g., filters and action menus, side menu icons?).

| **Icons**                                                                    | **Name**          | **Descriptions**                                                                                                                                                                                                                                                                                                |
| ---------------------------------------------------------------------------- | ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ![](../../.gitbook/assets/image22.png)                                       | Filter            | Click to access a list's filter panel and enter criteria to narrow the number of displayed items.                                                                                                                                                                                                               |
| ![](../../.gitbook/assets/image23.png)                                       | Expand Narrative  | Click to expand a narrative field for editing.                                                                                                                                                                                                                                                                  |
| ![](../../.gitbook/assets/image24.png)![](../../.gitbook/assets/image25.png) | Expand / Collapse | Click the Expand/Collapse arrows to display or hide fields.                                                                                                                                                                                                                                                     |

| ![](../../.gitbook/assets/image27.png)                                       | Notifications     | Click to view notifications.                                                                                                                                                                                                                                                                                    |
| ![](../../.gitbook/assets/image28.png)                                       | Priority Flag     | Click to flag a proforma as a priority.                                                                                                                                                                                                                                                                         |
| ![](../../.gitbook/assets/image29.png)                                       | Sort Order        | Click to sort a list in ascending or descending order.                                                                                                                                                                                                                                                          |
| ![](../../.gitbook/assets/image30.png)                                       | Attachment        | Click to manage attachments and notes.                                                                                                                                                                                                                                                                          |
| ![](../../.gitbook/assets/image31.png)                                       | Locked Status     | <p>This icon displays when a proforma is currently opened by a user in <a href="../../Proforma-Detail-View.md#proforma-detail-view">Proforma Detail view</a>.</p><p>Hover the cursor on the icon to view who locked the proforma. Locked proformas can be opened in read-only view.</p>                         |
