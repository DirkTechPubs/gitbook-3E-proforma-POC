### **Completed proformas**

Selecting the **Completed proformas** category displays all proformas for which the user is the Approver, and which are successfully submitted.

Proformas with the following statuses are available:

<td>-</td>

</tr>
<tr>
<td>Approved</td>

</tr>
<tr>

</tr>

</tr>
