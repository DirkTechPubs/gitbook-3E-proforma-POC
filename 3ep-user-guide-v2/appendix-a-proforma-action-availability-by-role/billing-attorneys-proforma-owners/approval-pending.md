### **Approval pending**

Selecting the **Approval pending** category displays all proformas for which the user is the owner, which are already submitted, but waiting for the approver’s review.

Proformas with the following statuses are available:

| **Status** | **Sub-status** | **Description** |
|----|----|----|
| **Approval pending** | \- | Proforma is waiting to be reviewed by the approver |

 

The following actions are available depending on the proforma status/sub-status:

| **Action** | **For what statuses/sub-statuses available** | **Description** |
|----|----|----|
| **Bill Preview** | Approval pending | Select this action to view a PDF version of the bill as it will appear to the client |
| **Print Proforma** | Approval pending | Select this action to view a PDF version of the proforma |
| **Reclaim** | Approval pending | Select this action to return a proforma back to editing after it was submitted |
